# Welcome to Triton Consulting Group's Data Science / Machine Learning Primer 

This primer acts as the foundational DSML curriculum for incoming associates to ensure technical machine learning and data science fundamentals are learned before being able to contribute to project work. 

## Topics Covered

- Finding and downloading datasets (Kaggle and other public sources)
- Data loading and project organization
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering and feature selection
- Building preprocessing pipelines
- Train/test splits and cross-validation
- Supervised machine learning fundamentals
- Logistic Regression, Decision Trees, Random Forests, Gradient Boosting, and Ensemble Methods
- Model evaluation and performance metrics
- Hyperparameter tuning with GridSearchCV
- Feature importance and model interpretation
- Error analysis and model improvement
- Saving trained models and machine learning best practices

## Prerequisites

- Basic Python programming
- Variables, loops, functions, and lists
- No prior Data Science or Machine Learning experience required

## Goal

By the end of this primer, you will be able to independently complete an end-to-end machine learning project, understand the reasoning behind each step of the data science process, and have the fundamentals necessary for working on a Tech Project with TCG!!!!
